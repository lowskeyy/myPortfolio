About Me

Halo Semuanya!. Saya Rizky Yanuar Irawan, saat saya membuat web ini, saya berusia 15 tahun. Saya berasal dari Bandung, Jawa Barat. Saya adalah anak ke-2 dari 2 bersaudara. Saya memiliki 1 kakak laki laki saya yang sekarang berusia 21 Tahun.


Saya adalah Siswa dari salah satu Sekolah Menengah Kejuruan (SMK) di Bandung, yaitu SMK Negeri 2 Bandung. Saat ini saya menginjak kelas 10 di jurusan PPLG (Pengembangan Perangkat Lunak dan Gim).






Projects Saya

Ini adalah Project Web Statis yang dibuat oleh Saya sendiri. Jika anda tertarik, anda bisa melihat project ini disini.

Ini adalah project membuat sebuah Aplikasi Matrix 2D, aplikasi ini ditugaskan oleh guru PPL dari UPI. Saya mengerjakan project ini sendiri walaupun kelompok saya berisikan saya dan 5 teman saya. Aplikasi ini dibuat menggunakan C++. Ini adalah tugas dalam materi Array, yaitu penjumlahan dan pengurangan sebuah array. Jika anda tertarik, anda bisa melihat project ini disini.

Ini adalah project membuat sebuah web sederhana untuk kelas saya, saya membuat web ini sendiri. Ini adalah project pertama saya saat saya sedang belajar menggunakan Animation dan Transition. Web ini berisikan mata pelajaran di kelas saya, nama guru yang mengajar di kelas saya, dan daftar tugas di setiap mata pelajaran. Jika anda tertarik, anda bisa melihat project ini disini.
